Gerador de Pokémon

Prova da Matéria de Desenvolvimento Web

Como funciona?

 O website utiliza uma API pública chamada PokéAPI.

 Quando pressionado o botão abaixo chamado "Gerar novo Pokémon", o seu navegador vai gerar um número aleatório e mandar uma requisição ao servidor da API.

 Um objeto com as informações do Pokémon escolhido será retornado se tudo estiver certo.

 Logo, as informações do Pokémon vão aparecer no site.
